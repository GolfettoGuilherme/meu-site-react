Esse site foi usado usando o tal do [Create React App](https://github.com/facebook/create-react-app).

## Como rodar? 

no diretórios do projeto, é só executar um

### `npm start`

Vai rodar o projeto em desenvolvimento.<br />
E seu navegador vai abrir sózinho o endereço [http://localhost:3000](http://localhost:3000)

A pagina é carregada "automagimaente" a cada ediçao.<br />
E todos erros vão aparecer no console. (e vão apareccer.)
